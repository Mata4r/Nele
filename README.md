# Nele Network Sniffer

Python sniffing tool inspired by Scapy.

# Features

- Standard packet sniffing

- HTTP & HTTPS sniffing

- TCP  sniffing
- UDP  sniffing
- ARP  sniffing
- SSH  sniffing
- ICMP sniffing
- Flag sniffing
- HTTP sniffing

# requirements

- Python 3.12+

# installation

- git clone https://github.com/Mata4r/Nele.git

- cd Nele/nele/main

- pip install -r requirements.txt

- python RunNele.py

- Run as Admin/sudo for full packet capture capabilities.
- Run Monitor mode before using Nele.
